ภาพรวมระบบ

Trash For Coin เป็นระบบมัดจำบรรจุภัณฑ์ขยะที่พัฒนาขึ้นเพื่อส่งเสริมการรีไซเคิลและลดขยะในชุมชน โดยใช้กลไกการมัดจำ 1 บาทต่อบรรจุภัณฑ์ 1 ชิ้น

วัตถุประสงค์หลัก

  • ลดขยะที่ไปหลุมฝังกลบ
  • เพิ่มอัตราการรีไซเคิล
  • สร้างแรงจูงใจในการคัดแยกขยะ
  • ลดต้นทุนการจัดการขยะ

ผลประโยชน์

  • ได้เงินคืนจากการทิ้งขยะ
  • ช่วยรักษาสิ่งแวดล้อม
  • ลดการใช้ทรัพยากรธรรมชาติ
  • เสริมสร้างจิตสำนึกสิ่งแวดล้อม

บทบาทผู้ใช้งาน

Root Admin

สิทธิ์สูงสุดในระบบ สามารถ:

  • จัดการผู้ใช้งานทุกระดับ
  • เข้าถึงข้อมูลทั้งหมด
  • ตั้งค่าระบบ
  • ดูรายงานทั้งหมด
Administrator

ผู้ดูแลระบบ สามารถ:

  • จัดการผู้ใช้งาน (ยกเว้น Root Admin)
  • จัดการสินค้าและหมวดหมู่
  • ดูรายงานทั้งหมด
  • จัดการคำสั่งซื้อ
Moderator

ผู้ดูแลเนื้อหา สามารถ:

  • จัดการสินค้าและหมวดหมู่
  • จัดการคำสั่งซื้อ
  • ดูรายงานส่วนใหญ่
  • ส่งออกข้อมูล
Member

สมาชิกทั่วไป สามารถ:

  • ดูและจัดการคำสั่งซื้อของตนเอง
  • ดูข้อมูลสินค้า
  • ส่งออกรายงานส่วนตัว
  • อัปเดตโปรไฟล์
Viewer

ผู้ดูข้อมูล สามารถ:

  • ดูข้อมูลทั่วไป
  • ดูสถิติระบบ
  • ดูรายการสินค้า
  • ไม่สามารถแก้ไขข้อมูล

เริ่มต้นใช้งาน

1. การสมัครสมาชิก

  1. เข้าสู่หน้าเว็บไซต์
  2. คลิกปุ่ม "สมัครสมาชิก"
  3. กรอกข้อมูล: ชื่อ, นามสกุล, อีเมล, รหัสผ่าน
  4. คลิก "สมัครสมาชิก"
  5. ระบบจะกำหนดบทบาทเป็น "Member" โดยอัตโนมัติ

2. การเข้าสู่ระบบ

  1. คลิกปุ่ม "เข้าสู่ระบบ"
  2. กรอกอีเมลและรหัสผ่าน
  3. คลิก "เข้าสู่ระบบ"
  4. ระบบจะพาเข้าสู่หน้าแรก
  5. ดูบทบาทของคุณที่มุมขวาบน
หมายเหตุสำคัญ

สำหรับการตั้งค่าครั้งแรก ผู้ดูแลระบบต้องรันคำสั่ง SQL ต่อไปนี้:

ALTER TABLE tbl_users ADD COLUMN role VARCHAR(50) DEFAULT 'member';
UPDATE tbl_users SET role = 'root_admin' WHERE email = 'your_admin_email@example.com';

ภาชนะพลาสติกประเภท PET

  • ขวดน้ำอัดลม
  • ขวดน้ำผลไม้
  • สามารถนำไปแปรรูปเป็นเม็ดพลาสติก
อลูมิเนียม

ภาชนะอลูมิเนียม

  • กระป๋องเครื่องดื่ม
  • กระป๋องเบียร์
  • ฝาขวดอลูมิเนียม
  • สามารถหลอมและขึ้นรูปใหม่ได้

ภาชนะแก้ว

  • ขวดแก้วยา
  • ขวดแก้วปรุงรส
  • วงจรการรีไซเคิลที่ยั่งยืน
วัสดุสำหรับเผา

ขยะที่เหมาะสำหรับเผาเพื่อผลิตพลังงาน

  • พลาสติกปนเปื้อนน้ำมัน
  • ถุงพลาสติกหลายชั้น
  • โฟมบรรจุภัณฑ์
  • กระดาษเคลือบพลาสติก
ขยะปนเปื้อน

ขยะที่ต้องจัดการพิเศษ

  • บรรจุภัณฑ์เปียก
  • ขยะปนเปื้อนของเหลว
  • ขยะปนเปื้อนน้ำมัน
  • ต้องมีการจัดการอย่างเหมาะสม

ระบบมัดจำ

การทำงานของระบบมัดจำ

  1. ซื้อสินค้า: ร้านค้าจะเก็บเงินมัดจำ 1 บาทต่อชิ้น
  2. ได้รับใบเสร็จ: พร้อมรหัส barcode สำหรับการคืนขยะ
  3. ใช้สินค้า: และเก็บบรรจุภัณฑ์ไว้
  4. คัดแยกขยะ: ตามประเภท 5 ประเภท
  5. นำไปทิ้ง: ที่ตู้รับขยะอัตโนมัติ
  6. สแกน barcode: จากใบเสร็จ
  7. ทิ้งขยะ: ในช่องที่ถูกต้อง
  8. รับเงินคืน: ทันที 1 บาทต่อชิ้น

ความต้องการของระบบ

  • เว็บเซิร์ฟเวอร์: Flask (Python)
  • ฐานข้อมูล: MySQL
  • Frontend: HTML, Bootstrap 5
  • ไลบรารี: mysql-connector-python, xhtml2pdf

โครงสร้างฐานข้อมูล

  • tbl_users: ข้อมูลผู้ใช้และบทบาท
  • tbl_category: หมวดหมู่ขยะ
  • tbl_products: สินค้าและบรรจุภัณฑ์
  • tbl_order: คำสั่งซื้อและการทิ้งขยะ
